Conduit.froude#

property Conduit.froude#

Get Link Results for Froude.

If Simulation is not running this method will raise a warning and return 0.

Returns:

Parameter Value

Return type:

float

Examples:

>>> from pyswmm import Simulation, Links
>>>
>>> with Simulation('tests/data/model_weir_setting.inp') as sim:
...     c1c2 = Links(sim)["C1:C2"]
...     for step in sim:
...         print(c1c2.froude)
0
1.2
1.5
1.9
1.2
